WPF-приложение для управления платежами с системой пользователей и аналитикой

Проект представляет собой десктопное приложение для учета и анализа платежей, разработанное на платформе WPF. Приложение включает систему авторизации с ролевой моделью, позволяющую разграничивать права доступа между администраторами и обычными пользователями.

Основной функционал включает управление платежами с возможностью их категоризации, фильтрации и сортировки. Реализована система пользователей с поддержкой загрузки фотографий профиля. Для визуализации данных предусмотрены различные типы диаграмм, а также возможность экспорта отчетов в форматы Excel и Word.

В приложении реализованы функции безопасности, включая хеширование паролей, защиту от перебора с помощью CAPTCHA и валидацию вводимых данных. Интерфейс поддерживает русскую локализацию и предоставляет интуитивно понятное управление данными.

Техническая реализация основана на .NET Framework 4.7.2 с использованием Entity Framework для работы с базой данных SQL Server. Архитектура приложения следует паттерну MVVM, что обеспечивает четкое разделение логики представления и бизнес-логики.

Проект демонстрирует комплексный подход к разработке бизнес-приложений с использованием современных технологий Microsoft и может служить основой для более сложных систем учета и управления данными.
